# Lesson Engine UI

Build the core learning system UI.

## Create Component

```
<LessonPlayer />
```

## Features

* Displays:

  * Title
  * Description
  * Code block (with copy button)
  * Command block (with copy button)
* Buttons:

  * Next
  * Previous
  * Mark Complete

## Copy Feature

Every code/command block must have:

* Copy button
* Toast feedback: "Copied!"

## Visual Layout

* Center panel
* Sidebar with:

  * Worlds
  * Regions
  * Progress %

## Output

Fully functional lesson navigation UI
